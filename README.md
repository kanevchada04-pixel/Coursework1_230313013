# S&P 500 Forecasting with Meta Prophet

This project explores time series forecasting of the S&P 500 using Meta Prophet in R.

The analysis includes:
- Exploratory time series analysis
- Moving averages
- Variance stabilisation with log transformation
- Forecasting with Prophet
- Comparison of different model configurations
